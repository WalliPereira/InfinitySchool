Crie um formulário HTML para registro de informações pessoais, incluindo campos para nome, data de nascimento, email e um campo de texto. 
Estilize o formulário usando CSS para centralizá-lo na página, aplicando propriedades de display, box model e border. 
Garanta que o formulário tenha uma borda arredondada, padding nos campos e margem entre eles.
Inclua também um botão de envio estilizado.
